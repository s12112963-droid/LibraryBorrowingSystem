# Library Borrowing System

This is a simple console application developed in C#.

## Features

- Add Physical Book
- Add EBook
- Register Member
- Borrow Book
- Return Book
- Search Book (Title, Author, ISBN)
- View Available Books

## Technologies

- C#
- .NET Console Application
- Object Oriented Programming (OOP)

## Structure

Models:
- Book
- PhysicalBook
- EBook
- Member

Services:
- LibraryService

Program:
- Console menu to interact with the system
