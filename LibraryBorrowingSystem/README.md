# Library Borrowing System 

A simple C# console application that simulates a library borrowing system.

## Features
- Book Management
  - Add Physical Book
  - Add EBook
  - View available books
- Member Management
  - Register new member (Id, Name, Email)
- Borrowing & Returning
  - Borrow only if the book is available
  - PhysicalBook: 14-day borrowing period
  - EBook: 7-day borrowing period
  - Return makes the book available again
- Search
  - Search by Title, Author, or ISBN

## Console Menu
1 - Add Physical Book  
2 - Add EBook  
3 - Register Member  
4 - Borrow Book  
5 - Return Book  
6 - View Available Books  
7 - Exit  

## How to Run
1. Open the solution in Visual Studio
2. Run the project (Start)
3. Use the console menu to interact with the system
